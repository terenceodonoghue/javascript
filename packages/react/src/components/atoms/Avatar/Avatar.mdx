# Avatar

An avatar component represents an object or entity, typically a user's profile picture.

```javascript
import { Avatar } from '@terenceodonoghue/react';
```

The `type` prop defines one of two styles: rounded (default) or squared. Standard [img](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/img) attributes are also supported, e.g. `alt`.

<Canvas of={AvatarStories.Type} />

Height and width can be customised by providing a number to the `size` prop.
