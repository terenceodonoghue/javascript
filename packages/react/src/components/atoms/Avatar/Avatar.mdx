# Avatar

An avatar component represents an object or entity, typically a user's profile picture.

## Usage

```javascript
import { Avatar } from '@terenceodonoghue/react';
```

Avatars can be created using standard [img](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/img) attributes, e.g. `alt` and `src`.

<Canvas of={AvatarStories.Usage} />

## Sizes

Height and width can be customised by providing a number (expressed as `rem`) to the `size` prop.

<Canvas of={AvatarStories.Sizes} />

## Variants

The `variant` prop defines one of two styles: rounded (default) or squared.
