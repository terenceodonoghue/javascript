# @terenceodonoghue/prettier-plugin-packagejson

A [Prettier](https://github.com/prettier/prettier) plugin to sort the keys of a package.json file.

## Installation

This package requires you to be [authenticated to GitHub Packages](../../README.md#setup).

```bash
npm install @terenceodonoghue/prettier-plugin-packagejson

# or...

yarn add @terenceodonoghue/prettier-plugin-packagejson
```
